# Trabajo22-5  La presente aplicación está destinada a ser utilizada en un bar y/o restaurante o similares. Dentro del software encontrará un inicio de sesión para usuario administrador y un botón para el ingreso de los empleados. 
El sistema contará con los apartados de Mesa, Mozo, Comanda, Detalle de la comanda. El sistema podrá controlar los estados de mesa, los pedidos realizados, etc.
